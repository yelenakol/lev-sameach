צור פרויקט ב-Google Cloud
פתח console.cloud.google.com → לחץ על "New Project" → תן לו שם (לדוגמה: lev-sameach) → Create.
2
הפעל את Google Drive API
בתפריט → "APIs & Services" → "Enable APIs" → חפש Google Drive API → Enable.
3
צור OAuth 2.0 Client ID
"APIs & Services" → "Credentials" → "Create Credentials" → "OAuth client ID"
Application type: Web application
Authorized JavaScript origins: null (או הכתובת שלך אם תשתמש ב-server)
לחץ Create ← תקבל Client ID.
4
הגדר OAuth Consent Screen
"OAuth consent screen" → User Type: External → מלא שם אפליקציה → Save.
ב-"Test users" הוסף את האימייל שלך.
